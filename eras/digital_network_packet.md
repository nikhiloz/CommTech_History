# Digital Era: Packet Switching

The foundational network technology that enabled the internet to exist.

## Packet Routing Flow
```text
  Source        Router 1       Router 2        Dest
    |              |              |              |
    |--[P1]------->|              |              |
    |              |--[P1]------->|              |
    |              |              |--[P1]------->|
    |--[P2]------->|              |              |
    |              |--[P2]------->|              |
    |              |              |--[P2]------->|
```

## Key Developments
- **Decentralization:** Unlike circuit-switched networks (like traditional telephone systems), packet switching breaks data into small packets, each routed independently.
- **Robustness:** If a link fails, packets take an alternative path, making the network highly resilient.
- **Efficiency:** Better utilization of network bandwidth compared to dedicated circuit connections.
