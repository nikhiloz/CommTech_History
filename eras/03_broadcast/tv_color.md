# Broadcast Era: Color Transition

The transition from black-and-white (monochrome) to color television was a monumental technical and marketing challenge.

## Technical Color Encoding Flow
```text
  [RGB Input] -> [Matrixing] -> [Luminance (Y)] --+--> [Modulator] --> [RF Out]
                                      |           |
                                      +--> [Chrominance (U,V)] --^
                                                |
                                        [Color Subcarrier]
```

## Key Developments
- **Compatibility:** The critical engineering hurdle was ensuring color broadcasts could still be received by existing monochrome TVs.
- **Encoding Techniques:** Sophisticated methods of encoding color information (chrominance) within the existing monochrome signal bandwidth (luminance).
- **Adoption:** Gradual rollout in the mid-20th century, significantly enhancing the visual richness and immersion of television content.
