# Modern Era: Cellular Evolution

The trajectory of wireless communication from mobile voice to global data networks.

## Technical Cellular Flow
```text
  [UE (Mobile Phone)] --(Radio Link)--> [Base Station (eNodeB)] -> [Core Network] -> [P-Gateway/Internet]
                                                |
                                          [Handover Mgt]
```

## Key Developments
- **1G (Analog):** First-generation mobile phones, localized and analog-based.
- **2G (GSM/CDMA):** Introduction of digital voice, SMS, and initial data capabilities.
- **3G:** Bringing true, albeit slow, mobile internet access to phones.
- **4G (LTE):** High-speed, IP-based mobile data, enabling mobile video, streaming, and advanced applications.
- **5G:** Focus on ultra-low latency, massive connectivity, and gigabit speeds.
