# Visual Communication (Pre-Electric)

## Signaling Flow
```text
  Beacon Site 1      Line of Sight       Beacon Site 2
       |                   |                   |
       |--- [Fire/Smoke]-->|                   |
       |                   |--- [Fire/Smoke]-->|
       |                                       |
    (Signal)                               (Observed)
```

Visual communication was the primary method for long-distance information transfer before electrical systems. These methods relied on line-of-sight and pre-arranged symbolic meanings.

## Key Methods
- **Smoke Signals:** Used by various civilizations (including Native American tribes and Ancient Chinese) to send messages over long distances. Different patterns and durations of smoke plumes conveyed specific alerts (e.g., danger, peace).
- **Fire Beacons:** High-altitude fires lit in sequence along chains of mountain tops or towers. Famous for the "warning of the enemy" systems used in many ancient empires.
- **Semaphore Towers:** A more sophisticated system using mechanical arms to represent letters or numbers. Signals were observed by telescopes at adjacent towers and relayed onwards, creating a "visual telegraph" network.
- **Flags and Heliographs:** Using flags (for ships or military) or sun-reflecting mirrors (heliographs) to transmit coded messages over shorter to medium distances.
