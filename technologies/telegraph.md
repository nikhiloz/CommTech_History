# Technology: Telegraph

The invention that launched the electrical communication revolution.

## Technical Telegraph Flow
```text
  [Key/Switch] --(Circuit Closed)--> [Battery Source] --(Current)--> [Electromagnetic Coil] -> [Armature/Sounder]
                                                                            |
                                                                     [Clicker Action]
```

## Key Developments
- **Mechanism:** Electromagnetic relays translating electrical pulses into physical action at a distance.
- **Key Concepts:** Electrical circuitry, binary coding (Morse code), signal propagation over long-distance wiring.
