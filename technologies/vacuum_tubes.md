# Technology: Vacuum Tubes

The enabling technology for early radio, TV, and computing.

## Technical Tube Amplification Flow
```text
  [Cathode (Heating)] -> [Electron Emission] -> [Grid Control (V_g)] -> [Anode (V_a)] -> [Amplified Output]
                                                        |
                                                 [Control Signal]
```

## Key Developments
- **Mechanism:** Using thermionic emission of electrons within a vacuum to control electrical current.
- **Key Concepts:** Amplification (crucial for weak radio signals), switching (early logic), signal generation.
- **Limitations:** Bulky, heat-intensive, prone to failure, eventually superseded by transistors.
